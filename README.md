# How it works
```bash
[Usage]: run.sh

[Parameters]:
 -d directory
 -t "title"
 -k "task description"
 -s "status" (open, close, breakfast, lunch, dinner) 

[Example]: ./run.sh -d example -k "my first task" -s open

[Help]: ./run.sh -h

[Interactive mode]: ./run.sh -i

[Warning]: Important to use 'space' remember of quotes "hello world"
```
